# Machine Learning II Mini Project

This project demonstrates different machine learning paradigms using Python, including supervised, unsupervised, reinforcement learning, and neural networks.

Implemented algorithms:
- SVM
- Q-learning
- K-means
- Neural Networks

---

## Results

- SVM Accuracy: **95%**
- Cross Validation Score: **98%**
- Neural Network Accuracy: **93%**
- K-means created **3 clusters**
- Q-learning improved decision making after training

- Python
